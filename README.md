# MnpnBot
Discord bot created in Ruby using DiscordRB.
Invite the bot with https://discordapp.com/oauth2/authorize?client_id=289471282720800768&scope=bot&permissions=0.

# Requirements
To use MnpnBot yourself you will need 4 gems:
- DiscordRB
- urban_dict
- json
- color
